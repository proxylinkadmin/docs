## May 2026

**Early Access launch**
- Freeware early access — full access, no credit card, no limits
- Ubuntu 22.04 server infrastructure
- Pi-hole DNS filtering for VPN clients
- MikroTik auto-config via SSH
- LAN auto-discovery with bulk proxy link creation

**New features**
- VNC file manager (browse, upload, download files via VNC terminal)
- Network automations — SSH triggers for device events and webhooks
- NOC dashboard for MSP operations centres
- Session recording for RDP and VNC (in addition to SSH)
- Wake-on-LAN through tunnel
- Asset inventory with CSV export
- NIS2 compliance portal

## April 2026

- Multi-VLAN support with additional subnets per tunnel
- Overlapping subnet handling via NETMAP
- Windows Server 2012 R2 support for deploy script
- ProxyLink push agent for remote Windows management
- VNC screen sharing (view-only for clients)
- 2FA (TOTP) support

## Earlier 2026

- Initial release with WireGuard tunnels, HTTP/HTTPS/TCP proxy links
- Browser SSH, RDP, VNC terminals
- MSP teams, client groups, access links
- Up/down monitoring with Slack/Teams/webhook alerts
- Custom domains
- Session recording (SSH)
