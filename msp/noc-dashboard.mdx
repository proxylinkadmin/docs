title: NOC Dashboard
description: Real-time status view across all client sites and devices.
---

The NOC dashboard gives a live overview of all tunnels and monitored services across all client groups. Designed for MSP operations centres running on a wall display.

## Access

Go to **More** → **NOC Dashboard**. Click **Full Screen** for wall display mode.

## What the NOC shows

- All client groups with live status
- Online / offline status for each tunnel
- Monitored services — up/down with last check time
- Offline device count per group

## Status thresholds

| Status | Meaning |
|--------|---------|
| Green | All devices online, all services up |
| Orange | One or more services down |
| Red | Tunnel offline or no handshake for 3+ minutes |

The dashboard auto-refreshes every 30 seconds.
