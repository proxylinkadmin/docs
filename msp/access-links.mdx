---
title: Access Links
description: Share device access with clients without requiring a ProxyLink account.
---

Access links let you share access to a specific device with anyone — no ProxyLink account required.

## Create an access link

1. Open a proxy link detail page
2. Click **Share** → **Create Access Link**
3. Optionally set an expiry date and stored credentials
4. Share the URL with the recipient

## How access links work

The recipient opens the URL and immediately accesses the device — no login, no ProxyLink account. If stored credentials are configured, they connect automatically.

## Security

- Access links are single-device — the recipient cannot navigate to other devices
- Expiry dates limit how long the link is valid
- Links can be revoked at any time from the proxy link detail page
- All access link sessions appear in the audit log
